Captura de 2 variables

  • realizar un software web que capture 2 valores enteros, que este software sea capaz de sumar, restar, multiplicar o dividir ambos valores y que pueda devolver el resultado de la operación seleccionada entre ambos numeros.

  • los 2 variables a ingresar, a y b, son valores enteros

  • el resultado también será un valor entero

  • al sumar, restar, multiplicar o dividir los valores, enviarlos a la variable del resultado
